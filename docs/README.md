# Documentation

## Learning Tasks

Work through these tasks in order to build your ASCII art generator:

1. [Task 1: Basic Setup](tasks/task-01-setup.md) - Get Go running with command line args
2. [Task 2: Read Files](tasks/task-02-read-files.md) - Load banner template files
3. [Task 3: Parse Characters](tasks/task-03-parse-characters.md) - Convert files to character maps
4. [Task 4: Render Single](tasks/task-04-render-single.md) - Display individual characters
5. [Task 5: Render Words](tasks/task-05-render-words.md) - Combine characters into words
6. [Task 6: Handle Newlines](tasks/task-06-handle-newlines.md) - Process `\n` for multiple lines
7. [Task 7: Basic Tests](tasks/task-07-basic-tests.md) - Write tests for your code
8. [Task 8: Organize](tasks/task-08-organize.md) - Clean up and structure code

## Banner Files

The banner template files in `assets/` (standard.txt, shadow.txt, thinkertoy.txt) contain ASCII art patterns for each character. Each character is 8 lines tall and characters are separated by empty lines.
